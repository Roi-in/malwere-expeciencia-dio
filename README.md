# malweres-expeciencia
Projeto educacional

Este repositório foi criado com o objetivo de estudo e compreensão de como alguns malwares funcionam na prática, para que seja possível aprender a identificar, prevenir e se defender de ataques reais.  

##  Ransomware 

O ransomware é um tipo de malware que sequestra os dados da vítima, aplicando criptografia com chave e impedindo o acesso aos arquivos. O mais alarmante é que pode ser implementado em poucas linhas de código, mas é capaz de causar danos catastróficos.

Características principais:
- Sequestra arquivos do usuário.
- Aplica criptografia usando uma chave (normalmente AES ou Fernet).
- Exibe uma mensagem de resgate exigindo pagamento.
- Somente quem tem a chave de descriptografia pode reverter o ataque, e essa chave fica nas mãos dos atacantes.

---

##  Keylogger 

O keylogger também é um malware simples de criar, e mesmo com poucas linhas de código pode capturar inúmeros dados confidenciais de uma pessoa sem que ela perceba.

Funcionamento:
- Captura todas as teclas digitadas pelo usuário.
- Salva os dados em um arquivo .txt, muitas vezes oculto.
- Pode ser programado para enviar as informações automaticamente por e-mail.

Esse tipo de ameaça mostra como uma simples coleta de teclado pode comprometer senhas, conversas, pesquisas e até dados bancários.

##  Defesa e Prevenção

Durante o estudo, ficou claro que o ponto mais importante na segurança é o próprio usuário. Um usuário consciente evita muitos problemas. Além disso, existem várias medidas essenciais de proteção:

Medidas importantes:
- Conscientização do usuário, a primeira linha de defesa.
- Antivírus e firewall sempre ativos e atualizados.
- Máquinas virtuais (VMs) para testes e análise segura.
- Backups frequentes para minimizar os danos em caso de ataque.
- Evitar downloads de fontes desconhecidas.

## Conclusão

Esse estudo foi fundamental para entender como os ataques realmente acontecem. Compreender o funcionamento de um malware ajuda diretamente a entender como se proteger.  
Foi surpreendente perceber como certos malwares são relativamente fáceis de implementar, e isso reforça uma ideia importante:

No final, a maior proteção é um usuário consciente.
